# LuckyDraw
A modern Lucky Draw application built with Streamlit to randomly select employees who will win the Umrah Trip.

The application displays employee names in an animated rolling effect before announcing the winner, creating an engaging and exciting experience during company events.

Features

Animated random name rolling
Automatic winner selection
Removes winners to avoid duplicates
Displays Employee Name + ID
Responsive design (Desktop & Mobile)
Custom background support
Simple Excel data source
